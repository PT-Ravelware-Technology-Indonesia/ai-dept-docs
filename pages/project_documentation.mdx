Project Documentation di sini fokus ke **big picture** tiap project — bukan duplikasi README atau changelog.

Pembagiannya gini:

| Lapisan | Tempat | Isi |
|---------|--------|-----|
| **README** | Repository GitHub | Setup, cara run, dependencies |
| **Project Documentation** | Halaman ini | Arsitektur, integrasi, konteks bisnis, keputusan teknis |
| **Changelog** | GitHub Releases | Bug fix, update fitur, tracking tiap rilis |

Panduan lengkap soal dokumentasi project ada di <Link to="/monitoring_iteration/documentation_guide">Documentation Guide</Link>.

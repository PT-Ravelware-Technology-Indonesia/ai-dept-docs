---
title: Cloud Deployment
---

Pada dasarnya Cloud Deployment mengacu pada proses deploy model AI ataupun aplikasi di cloud, seperti AWS, GCP, atau Azure.
Cara paling straightforward untuk melakukan Cloud Deployment adalah dengan meng-*containerize* aplikasi kita menggunakan <Link to="../home/tools/docker">Docker</Link>, lalu deploy container tersebut ke cloud provider yang kita gunakan.

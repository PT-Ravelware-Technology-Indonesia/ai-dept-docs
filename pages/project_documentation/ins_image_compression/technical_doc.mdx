## Project Overview
Image Compression INS merupakan aplikasi web fullstack untuk mengotomasi proses kompresi gambar .jpg menggunakan ffmpeg dan dibuat menggunakan [FastAPI](https://fastapi.tiangolo.com/) backend and static frontend. dengan *core features* berikut:

- Directory scanning untuk file .jpg
- Image compression menggunakan FFmpeg
- User authentication and management
- Dashboard dengan metrics
- Job Tracking
- File backup and scheduling

Tech stack utama: [SQLModel](https://sqlmodel.tiangolo.com/) untuk ORM, [Alembic](https://alembic.sqlalchemy.org/) untuk migrasi database, dan [uv](https://docs.astral.sh/uv/) untuk dependency management.


## Architecture Overview
Program image compression merupakan web app yang berjalan sebagai windows service di server INS.

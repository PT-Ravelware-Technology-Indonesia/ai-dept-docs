## Project Overview
Project ini merupakan project automasi image compression di PT INS. Tujuannya adalah untuk mengurangi ukuran file gambar tanpa mengorbankan kualitas visual secara signifikan dan mengotomasi prosesnya.

## Architecture Overview
Program image compression merupakan web app yang berjalan sebagai windows service di server INS.

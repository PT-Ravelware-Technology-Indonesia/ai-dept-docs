## Project Overview

People Counting merupakan aplikasi web untuk mendeteksi dan menghitung jumlah orang secara otomatis menggunakan computer vision. Dengan *core features* berikut:

- Deteksi orang real-time dari kamera/CCTV
- Visual counting dashboard
- Riwayat dan tren data per periode
- Multi-camera support
- Export laporan

---

## Konteks Bisnis

People Counting dibuat untuk menjawab kebutuhan monitoring dan analisis jumlah pengunjung di suatu area. Tujuan utamanya:

- **Monitoring real-time** — mengetahui jumlah orang di suatu area secara langsung
- **Data-driven decision** — analisis tren pengunjung untuk kebutuhan operasional
- **Otomatisasi** — menggantikan manual counting yang rentan error

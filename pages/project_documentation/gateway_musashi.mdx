## Project Overview

Gateway Musashi merupakan program gateway yang diprogrm untuk kebutuhan smart oil station Musashi. Program v2.0.0 merupakan hasil rewrite dari program sebelumnya dengan mengganti sistem verifikasi menjadi token-basesd.

Tech stack utama: Fastapi, [SQLModel](https://sqlmodel.tiangolo.com/) untuk ORM, [Alembic](https://alembic.sqlalchemy.org/) untuk migrasi database, dan [uv](https://docs.astral.sh/uv/) untuk dependency management.

## Alur Proses

### 1. Token Generation and Validation
Token digenerate dari sisi software, semua validasi pembuatan token dilakukan di software, gateway hanya menerima dan menyimpan token yang sudah digenerate. Token digunakan untuk autentikasi setiap request dari HMI ke gateway, memastikan hanya request yang valid yang diproses.

<Mermaid
    config={{look: "handDrawn"}}
    chart={
        `sequenceDiagram
    Software->>+Gateway: Send tokens (Bulk)
    HMI->>+Gateway: Input token
    activate Gateway
    Gateway->>Software: Send data and status 'start'
    deactivate Gateway
    HMI->>Gateway: Finished dispensing oil, send actual oil taken
    Gateway->>Software: Send data and status 'finish'

        `
    }
/>
:::note
Catatan penting:
- Token Expiration dihandle masing-masing antara software dan gateway, tidak saling bergantung ataupun sync.
- Validasi token terjadi sepenuhnya di program gateway tanpa perlu komunikasi dengan software.
:::

## Deployment

Aplikasi berjalan sebagai **Windows Service** menggunakan NSSM. Ini memastikan:

- Aplikasi **auto-start** pas server nyala
- **Restart otomatis** kalo crash

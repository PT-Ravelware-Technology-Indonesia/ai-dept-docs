## Inisialisasi Project

Inisialisasi project baru menggunakan <Link to="../home/tools/uv">uv</Link> dan konfigurasi <Link to="../home/tools/ruff">ruff</Link>.
Gunakan `kebab-case` untuk nama project. contoh: `smart-scale`.

Struktur project menyesuaikan kebutuhan, namun berikut adalah beberapa project yang sering digunakan atau berulang.

### Fastapi Service
Fastapi service merupakan sistem dimana kita menyediakan service endpoint untuk model AI kita dan integrasi dengan software melalui HTTP.
<Mermaid
    config={{look: "handDrawn"}}
    chart={
        `flowchart LR
    ai-service[Fastapi Service]
    backend-service[Backend/Frontend Service]
    ai-service <--HTTP--> backend-service`
    }
/>

Gunakan template berikut untuk membuat Fastapi service baru <Link to="https://github.com/AryaElRumy/fastapi-edge-template.git">Fastapi Service Template</Link>

### Gateway Service
Gateway Service merupakan sistem dimana kita menerima data dari sensor ataupun berkomunikasi dengan microcontroler melalui protokol MQTT.
<Mermaid
    config={{look: "handDrawn"}}
    chart={
        `
        flowchart LR
    ai-service[AI Service]
    mqtt-broker[MQTT Broker]
    sensors[sensors]
    ai-service<-->mqtt-broker
    sensors-->mqtt-broker
        `
    }
/>

Gunakan template berikut untuk membuat Fastapi service baru <Link to="https://github.com/AryaElRumy/bridge-gateway-template.git">Gateway Service Template</Link>
